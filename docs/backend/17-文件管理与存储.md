# 17 - 文件管理与存储

## 适用场景
管理后台的文件上传/下载/预览——用户头像、文档附件、图片素材等。

## 关联原则
- 原则5：输入验证 — 文件类型白名单、大小限制、文件名安全处理
- 原则6：最小权限 — 文件访问需鉴权（非公开URL）

## 核心规则
1. **文件类型白名单**（不允许的扩展名直接拒绝）
2. **文件大小限制**（前端+后端双重校验）
3. **文件名安全处理**（防路径穿越、统一命名）
4. **文件存储可切换**（本地开发 / S3生产）
5. **文件访问经过鉴权**（非公开URL直链）

### 步骤 2：文件 Model

文件：`apps/common/models.py`

```python
import uuid
from django.db import models
from apps.common.base_model import BaseModel


def upload_to(instance, filename):
    """生成唯一文件名 + 按日期分目录"""
    ext = filename.rsplit('.', 1)[-1].lower()
    date_path = instance.created_at.strftime('%Y/%m') if instance.created_at else 'temp'
    return f'uploads/{date_path}/{uuid.uuid4().hex}.{ext}'


class FileRecord(BaseModel):
    """文件记录 — 文件实体元数据，实体文件存储在 MEDIA_ROOT"""

    original_name = models.CharField(max_length=255, verbose_name="原始文件名")
    stored_name = models.CharField(max_length=255, verbose_name="存储文件名")
    file = models.FileField(upload_to=upload_to, verbose_name="文件")
    file_size = models.BigIntegerField(verbose_name="文件大小(bytes)")
    mime_type = models.CharField(max_length=100, verbose_name="MIME类型")
    file_type = models.CharField(max_length=20, choices=[
        ('image', '图片'), ('document', '文档'), ('archive', '压缩包'),
        ('other', '其他'),
    ], verbose_name="文件分类")
    uploaded_by = models.ForeignKey(
        'rbac.User', on_delete=models.SET_NULL, null=True, verbose_name="上传者"
    )

    objects = BaseManager()

    class Meta:
        db_table = 'common_files'
        verbose_name = '文件'

    @property
    def extension(self):
        return self.original_name.rsplit('.', 1)[-1].lower() if '.' in self.original_name else ''
```

### 步骤 3：文件上传 Service（含安全校验）

文件：`apps/common/services/file_service.py`

```python
import os
import magic
from django.conf import settings
from django.core.files.uploadedfile import UploadedFile
from apps.common.models import FileRecord
from apps.common.exceptions import ErrorCode


class FileService:
    """文件管理服务"""

    def upload(self, file: UploadedFile, uploaded_by_id: int) -> dict:
        """上传文件（含安全校验）"""
        # 1. 扩展名校验
        ext = file.name.rsplit('.', 1)[-1].lower() if '.' in file.name else ''
        allowed = self._get_allowed_extensions()
        if ext not in allowed:
            raise ErrorCode.FILE_TYPE_DENIED.as_exc(
                detail=f"不支持的格式 .{ext}，允许：{', '.join(allowed)}"
            )

        # 2. 大小校验
        max_size = settings.FILE_UPLOAD_MAX_MEMORY_SIZE
        if file.size > max_size:
            raise ErrorCode.FILE_TOO_LARGE.as_exc(
                detail=f"文件最大 {max_size // 1024 // 1024}MB"
            )

        # 3. 文件名安全处理（防路径穿越）
        safe_name = os.path.basename(file.name)

        # 4. 检测真实 MIME 类型（防伪造扩展名）
        mime_type = magic.from_buffer(file.read(2048), mime=True)
        file.seek(0)

        # 5. 保存文件
        record = FileRecord.objects.create(
            original_name=safe_name,
            stored_name=safe_name,  # upload_to 会自动重命名
            file=file,
            file_size=file.size,
            mime_type=mime_type,
            file_type=self._classify_type(ext),
            uploaded_by_id=uploaded_by_id,
        )

        return {
            'id': record.id,
            'url': record.file.url,
            'name': record.original_name,
            'size': record.file_size,
        }

    def _get_allowed_extensions(self) -> set:
        return {
            ext
            for types in settings.ALLOWED_FILE_TYPES.values()
            for ext in types
        }

    def _classify_type(self, ext: str) -> str:
        for file_type, exts in settings.ALLOWED_FILE_TYPES.items():
            if ext in exts:
                return file_type
        return 'other'
```

### 步骤 4：文件上传 ViewSet

```python
from rest_framework import viewsets
from rest_framework.response import Response
from rest_framework.permissions import IsAuthenticated


class FileUploadViewSet(viewsets.GenericViewSet):
    """文件上传接口"""
    permission_classes = [IsAuthenticated]

    def create(self, request):
        file = request.FILES.get('file')
        if not file:
            return Response(
                {"error": {"code": "NO_FILE", "message": "未选择文件"}},
                status=400,
            )
        result = FileService().upload(file, request.user.id)
        return Response({"data": result}, status=201)
```

### 步骤 5：文件访问鉴权（非公开 URL）

```python
# 不直接暴露 MEDIA_URL，通过鉴权视图代理
class FileDownloadViewSet(viewsets.GenericViewSet):
    permission_classes = [IsAuthenticated]

    def retrieve(self, request, pk=None):
        record = FileRecord.objects.filter(id=pk).first()
        if not record:
            raise ErrorCode.NOT_FOUND.as_exc()

        response = HttpResponse(record.file, content_type=record.mime_type)
        response['Content-Disposition'] = f'inline; filename="{record.original_name}"'
        return response
```

## 反模式（禁止）
- ❌ 前端直接校验文件类型（可绕过） → 后端 MIME 检测
- ❌ 文件名直接拼路径 → `os.path.basename` + uuid 重命名
- ❌ 文件 URL 公开可访问 → 通过鉴权视图代理
- ❌ 大文件同步上传阻塞请求 → 异步上传（Celery 任务）
