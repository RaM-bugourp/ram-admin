# 09 - 审计与日志开发

## 适用场景
实现操作审计（谁在什么时候做了什么）、结构化日志输出。

## 关联原则
- 原则10：可观测性 — JSON 结构化日志，含 requestId/userId/耗时
- 原则7：一致性优先 — 审计日志与业务操作在同一事务中

## 核心规则
1. **审计日志与业务事务原子化**（同在一个 `@transaction.atomic` 中）
2. **日志 JSON 格式**（不用文本日志）
3. **包含 request_id**（全链路追踪）
4. **敏感字段脱敏**（密码、手机号、身份证号）

## 开发步骤

### 步骤 1：审计模型

文件：`apps/audit/models.py`

```python
from django.db import models
from apps.common.base_model import BaseModel

class AuditLog(BaseModel):
    """审计日志 - 只增不改不删"""

    ACTION_TYPES = [
        ('create', '创建'), ('update', '更新'), ('delete', '删除'),
        ('login', '登录'), ('logout', '登出'),
        ('role_change', '角色变更'),
        ('permission_change', '权限变更'),
    ]

    user = models.ForeignKey('rbac.User', on_delete=models.PROTECT, null=True)
    user_name = models.CharField(max_length=150, default="")
    action = models.CharField(max_length=50, choices=ACTION_TYPES)
    resource = models.CharField(max_length=100, verbose_name="操作资源")
    resource_id = models.IntegerField(null=True, verbose_name="资源ID")
    detail = models.JSONField(default=dict, verbose_name="详情")
    ip_address = models.GenericIPAddressField(null=True)
    request_id = models.CharField(max_length=50, default="")

    class Meta:
        db_table = 'audit_logs'
        ordering = ['-created_at']
        indexes = [
            models.Index(fields=['user', 'action']),
            models.Index(fields=['resource', 'resource_id']),
            models.Index(fields=['created_at']),
        ]
```

### 步骤 2：审计 Service

文件：`apps/audit/services.py`

```python
import logging
from apps.audit.models import AuditLog

logger = logging.getLogger('audit')

class AuditService:
    """审计服务 - 记录谁在什么时候做了什么"""

    @classmethod
    def log(cls, **kwargs):
        """通用审计记录"""
        try:
            AuditLog.objects.create(**kwargs)
        except Exception as e:
            logger.error("Audit log write failed", exc_info=e, extra={
                'action': kwargs.get('action'),
                'user_id': kwargs.get('user_id'),
            })

    @classmethod
    def log_role_permission_change(cls, role_id: int, permission_ids: list):
        """记录角色权限变更"""
        from apps.rbac.repositories import RoleRepository
        role = RoleRepository().get_by_id(role_id)
        cls.log(
            user_id=None,  # 由调用方传入
            user_name='system',
            action='permission_change',
            resource='Role',
            resource_id=role_id,
            detail={
                'role_name': role.name if role else 'unknown',
                'permission_ids': permission_ids,
            }
        )
```

### 步骤 3：日志工具函数

文件：`apps/common/logger.py`

```python
import logging
import json
from datetime import datetime, timezone

class AuditLogger:
    """结构化审计日志记录器"""

    def __init__(self, logger_name='audit'):
        self.logger = logging.getLogger(logger_name)

    def log_business(self, action: str, **extra):
        """记录业务操作"""
        self.logger.info(f"Action: {action}", extra={
            'action': action,
            'timestamp': datetime.now(timezone.utc).isoformat(),
            **extra
        })

    def log_error(self, error_code: str, message: str, **extra):
        self.logger.error(message, extra={
            'error_code': error_code,
            **extra
        })

# 全局实例
audit_logger = AuditLogger()
```

### 步骤 4：敏感信息脱敏

```python
import re

def mask_sensitive(data: dict) -> dict:
    """脱敏敏感字段"""
    result = dict(data)
    SENSITIVE_FIELDS = ['password', 'password_hash', 'token', 'secret', 'id_card']
    for field in SENSITIVE_FIELDS:
        if field in result:
            result[field] = '***MASKED***'
    result['email'] = re.sub(r'(.).*(@.*)', r'\1***\2', result.get('email', ''))
    return result
```

## 日志级别使用规范
| 级别 | 场景 | 示例 |
|------|------|------|
| INFO | 正常业务流程 | 用户登录、创建订单 |
| WARNING | 异常但可恢复 | 登录失败、权限不足 |
| ERROR | 需要人工介入 | 数据库连接失败、支付异常 |
| DEBUG | 开发调试 | SQL 查询、请求参数 |
