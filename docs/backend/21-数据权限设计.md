# 21 - 数据权限设计

## 适用场景
实现行级数据权限控制——不同用户看到不同范围的数据（本部门 / 本部门+子部门 / 全部）。

## 关联原则
- 原则6：最小权限 — 默认只能看自己的数据，需要显式授权才能扩大范围
- 原则3：关注点分离 — 数据权限在 ViewSet 的 `get_queryset` 中实现，不在 Service

## 核心规则
1. **数据权限在 ViewSet.get_queryset() 层实现**（不在 Service 也不在 Repository）
2. **部门树用 MPTT**（`django-mptt`），支持高效子树查询
3. **数据权限范围定义**：self / department / department_and_children / all
4. **数据权限与接口权限独立**（接口权限控制能不能调API，数据权限控制能看到什么数据）

## 数据权限范围定义

| 范围 | 含义 | QuerySet 过滤 |
|------|------|-------------|
| `self` | 仅自己的数据 | `filter(owner_id=user.id)` |
| `department` | 本部门 | `filter(department_id=user.department_id)` |
| `dept_and_children` | 本部门+所有子部门 | `filter(department_id__in=subtree_ids)` |
| `all` | 全部数据 | 不过滤 |

```python
from mptt.models import MPTTModel, TreeForeignKey


class Department(MPTTModel, BaseModel):
    """部门 — MPTT 树结构"""
    name = models.CharField(max_length=100, verbose_name="部门名称")
    code = models.CharField(max_length=50, unique=True, verbose_name="部门编码")
    parent = TreeForeignKey(
        'self',
        on_delete=models.CASCADE,
        null=True,
        blank=True,
        related_name='children',
        verbose_name="上级部门",
    )

    objects = BaseManager()

    class Meta:
        db_table = 'rbac_departments'
        verbose_name = '部门'

    class MPTTMeta:
        order_insertion_by = ['code']

    def get_all_child_ids(self):
        """获取所有子部门ID（含自身）"""
        return (
            self.get_descendants(include_self=True)
            .values_list('id', flat=True)
        )
```

### 步骤 2：ViewSet 中实现数据权限

文件：`apps/rbac/views/user_views.py`

```python
from rest_framework import viewsets
from apps.rbac.models import User, Department


class BaseDataScopedViewSet(viewsets.GenericViewSet):
    """数据权限基类 — 所有需要行级权限的 ViewSet 继承此类"""

    data_scope_field = 'department_id'   # 子类可覆盖
    owner_field = 'created_by_id'        # 数据所有者字段

    def get_data_scope(self):
        """获取当前用户的数据权限范围"""
        user = self.request.user
        if user.is_superuser:
            return 'all'
        # 从用户角色中取最大数据权限范围
        scope = user.get_max_data_scope()  # 返回 'self' / 'department' / 'dept_and_children' / 'all'
        return scope

    def apply_data_scope(self, queryset):
        """根据数据权限范围过滤 queryset"""
        scope = self.get_data_scope()
        user = self.request.user

        if scope == 'self':
            return queryset.filter(**{self.owner_field: user.id})
        elif scope == 'department':
            return queryset.filter(**{self.data_scope_field: user.department_id})
        elif scope == 'dept_and_children':
            if not user.department:
                return queryset.none()
            dept_ids = user.department.get_all_child_ids()
            return queryset.filter(**{f'{self.data_scope_field}__in': dept_ids})
        else:  # 'all'
            return queryset

    def get_queryset(self):
        queryset = super().get_queryset()
        return self.apply_data_scope(queryset)


class UserViewSet(BaseDataScopedViewSet):
    """用户管理 ViewSet — 含数据权限"""
    queryset = User.objects.active()
    data_scope_field = 'department_id'
    owner_field = 'created_by_id'

    # 普通方法用 get_queryset() 自动过滤
    # 列表/详情/更新/删除 都会通过数据权限过滤
```

### 步骤 3：字段级权限（Serializer 动态字段）

某些字段（如薪资）仅特定角色可见：

```python
class UserOutputSerializer(serializers.Serializer):
    id = serializers.IntegerField()
    username = serializers.CharField()
    email = serializers.CharField()
    department_name = serializers.CharField()
    salary = serializers.DecimalField(max_digits=10, decimal_places=2)

    def to_representation(self, instance):
        data = super().to_representation(instance)
        request = self.context.get('request')
        if not request or not request.user.has_perm('user.view_salary'):
            data.pop('salary', None)  # 移除敏感字段
        return data
```

### 步骤 4：数据权限与分页/过滤的组合

```python
class UserViewSet(BaseDataScopedViewSet, 
                  mixins.ListModelMixin,
                  mixins.RetrieveModelMixin):
    queryset = User.objects.active()
    filterset_fields = ['is_active', 'department_id']  # 额外过滤
    search_fields = ['username', 'email']               # 搜索
    ordering_fields = ['created_at', 'username']        # 排序

    def get_queryset(self):
        # 数据权限 + 过滤 + 搜索 + 排序 组合链
        qs = super().get_queryset()  # BaseDataScopedViewSet 已应用数据权限
        # DRF filter/search/ordering 自动叠加在数据权限过滤后的 qs 上
        return qs
```

## 反模式（禁止）
- ❌ 数据权限写在 Service 层 → 在 ViewSet.get_queryset 实现
- ❌ 部门树用 `parent_id` 自关联 + 递归查询 → 用 MPTT
- ❌ 硬编码数据权限范围 → 从角色/配置中读取
- ❌ 数据权限和接口权限混在一起 → 分离：接口权限=permission_classes，数据权限=get_queryset
